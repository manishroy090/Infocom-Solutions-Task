### Queue Configuration (Required)

This project uses **Laravel Queues** for background processing (Excel import).

Make sure your queue connection is configured in `.env`:

```env

```

Run queue migrations if not already done:

```bash

php artisan migrate
```

Start the queue worker **before performing Excel import**:

```bash
php artisan queue:work
```

## ğŸ” Authentication Flow

* JWT-based authentication
* Token is generated by Laravel and stored in **HTTP-only cookies**
* **JWT is handled only on the backend**
* Frontend does not store or manage tokens
* Cookies are automatically sent with API requests
* Improves security against XSS attacks
